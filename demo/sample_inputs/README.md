# Sample Inputs

This directory contains the stable offline inputs used by the competition demo.

- `sample_video_request.json`: the ingest request passed to the Harness demo.
- `../sample_videos.json`: video metadata adapter source.
- `../sample_transcripts/BVDEMOHARNESS01.json`: timestamped transcript source.

These files are intentionally local and deterministic so the demo can run
without Bilibili login, external ASR, or network access.
